# MMM-iCutz

This module displays a camera feed, from a PiCamera, on the Magic Mirror.

## Dependencies

python3

## How to use

Add the following to your **config/config.js**

```
{
    module: "MMM-iCutz",
    position: "middle_center"
},
```
